
# LaunchiCube

A simple multi-instance launcher for ClassiCube, inspired by MultiMC, and made in Python

## Requirements

- Recomended Python Version: 3.12
- requests

## Features

游릴 = Done
游릳 = In Development
游린 = Not In Development

- 游릳 Instances
	- 游릴 Multi-Instances
   	- 游릴 Adding Instances
	- 游릳 Custom Instance Icons
	- 游린 Deleting Instances
	- 游린 Editing Instances
	- 游린 Plugin Manager
   	- 游린 Multible Instances Running
  		- 游린 Stop Button
   	  	- 游린 Management of Multible Instances
	- 游린 Custom Clients
- 游릳 Misc
	- 游릳 Mac, Linux, and 32bit Support
  		- 游릳 Mac Downloads
   		- 游릳 Linux Downloads
   		- 游릴 32bit Downloads For All OSes
	- 游린 Account Switcher
	- 游린 Settings
	- 游린 Themes
